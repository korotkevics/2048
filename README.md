# Play 2048 - Hexagonal Architecture Edition

A high-performance, feature-rich 2048 game featuring professional AI strategies (Expectimax and LLM-based), persistent session management, and a modern React frontend, all built using clean Hexagonal Architecture principles in Java/Spring Boot.

To build and run the entire application stack including the PostgreSQL database and AI services, ensure you have Docker and Docker Compose installed, then execute `docker compose up --build` from the root directory; the game will be available at `http://localhost:85`.
